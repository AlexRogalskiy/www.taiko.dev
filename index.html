Reliable Browser Automation

Taiko is a open source Node.js library for
automating chromium browsers

Everything you need for reliable end to end testing

Taiko Features

Smart Selectors

Write scripts by looking at a web page, without inspecting it`s source code. Create reliable tests that don`t break with changes in the web page's structure.

Implicit Waits

An intuitive API listens to actions and automatically waits for them to complete before moving on to the next one. Scripts are free of explicit waits that reduce 'flakiness'.

Interactive Recorder

An Interactive Recorder that's a REPL to write test scripts. You can use Taiko's API to control the browser from REPL.

Request & Response

Setting up test infrastructure and test data is hard. With the intercept API, you don`t have to set up mock servers, or replace URLs in tests to point to test instances.
